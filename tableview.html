Table View

The Table View is one of primary features of CloudTrial Viewer, it is the place you can see the events that have been loaded, inspect individual events, filter the events and view charts based on the loaded events.

The majority of the screens provide a Table View aspect allowing you to see a drilled down representation of the loaded events, for example on the Service Overview screen you can click a service and the table only shows the events for that service.

Clicking on the Table header you are able to select the columns that you wish to see, and hide those that you don't.

The Simple Table View screen consists of a Table view that shows all the events that have been loaded.

Export Table

Above the table to the right of the filter bar is the Export Table button. This feature allows the contents of the table to be save as a CSV formatted file. Only the visible columns will be save.

Side Bar

Part of the Table View is the Side bar. This area provides more information about the events loaded and individual events selected.

Save

The save button will create an image of the currently selected side bar function opening a Save Dialog so you can choose where to save it too. The image created will be a .PNG file.

Event JSON

Sometime the information your need to look at doesn't fit well into a table format. Clicking any row in the table will open the side bar (if not already visible) and present the original JSON that formed the CloudTrail Event.

Event Tree

This feature shows a tree representation of the Event.

Chart Stats

The Chart Stats feature shows all the events contained in the table as a chart. This chart can show either the Top 5 or Top 10 events that match a criteria. You change the criteria by selecting from the Source menu what Event parameter you want too see. You can change the type of chart, it's orientation. Below the Chart is a key showing the data represented in the chart.
