User Overview

What IAM Users are the most active?


The User Overview screen shows all the IAM Users and IAM roles that have made API calls, along with the number of API calls made.

Clicking either a Users or role will populate the right hand side of the screen. At the top there is a Service Overview panel that breaks down the API calls made by Service and at the bottom is a Table view contains all the API calls made.
