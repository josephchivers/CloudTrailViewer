How the application works.



CloudTrail viewer is a Java 7 Swing application.

The source code in this repository is presented as a Java Maven project, the easiest way of compiling and running the code from source is to use your favourite Java Ide, create a project that is Maven aware and copy the downloaded code into that project.

Netbeans

If you use the Netbeans IDE then you can create a project using the New Project wizard that is a Java Application in the Maven Category. Once you have created the project, simple download a zip, or clone this repository into that project. Depending on your current development environment you may notice a warning triangle against the project name, this will probably mean that you don't have all the dependencies on your machine, all you have to do is Right-Click on the project, select Resolve Project Problems, and then select Resolve from the displayed dialog.

Artifacts

Currently the Maven build process creates two artificats:

  • OS X Application (.app)
  • Windows Application (.exe)

Neither of these are signed in any shape or form so end users might have problems running them if they are not aware of the commands to run unsigned applications.
