CloudTrail Viewer

Service Overview

What AWS services are the most active.

The Service Overview screen is normally the first screen to load as Events are being loaded.

As events are loaded into CloudTrail Viewer a new Service component will be added to the screen. This component shows on the left the number of API calls made by AWS in monitoring your resources, and the number of API calls made by your IAM Users / Roles on the right. The total number of API calls made to the service is shown at the bottom.

Clicking one of the Services will show a Table View to appear at the bottom of the screen containing just the events belonging to that service. All the functionality provided by the Table View feature is available.
