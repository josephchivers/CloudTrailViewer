CloudTrail Viewer

Service Overview

What AWS services are the most active.



The Service Overview screen is normally the first screen to load as Events are being loaded. As events are loaded into CloudTrail Viewer a new Service detail panel will be added to the screen.



Each Service detail panel consists of the following items:

  • The service name at the top of the box
  • The total number of API Calls that were loaded at the bottom of the box
  • The main section of the box breaks down the API Calls into two areas
    • On the left (the blue colour) are the total number of API calls made by AWS on your behalf while monitoring your resources
    • On the right (the greeny colour) are the total number of API calls made your IAM User/Group/Roles
    • Under each section are the Peak Events per Second/Minute/Hour

Clicking one of the Total number will cause a Table View to appear at the bottom of the screen (if it's not already visible) containing just the events belonging to that particular count. All the functionality provided by the Table View feature is available.
